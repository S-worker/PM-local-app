    📘 Manuel Utilisateur - Gestionnaire de Projets Local Pro

    Bienvenue dans votre outil de pilotage de projet local et sécurisé. Cette application s'exécute entièrement dans votre navigateur tout en sauvegardant vos données directement dans votre écosystème local (OneDrive ou disque local). Pas de base de données complexe, pas de cloud tiers : vous êtes maître de vos données.

    🤖 La dimension "Intelligence Artificielle"

    Cet outil a été conçu avec l'aide de l'IA (approche vibe-coding) pour garantir un code léger et optimisé. Il n'embarque aucune IA générative en tâche de fond. Tous les calculs (rétro-planning, chemins critiques, agrégation des données) sont gérés de manière transparente par des algorithmes mathématiques stricts et déterministes. Vos données ne sont jamais envoyées à un modèle (LLM) externe.

    👑 0. Gestion des Profils & Usage Direction (Steerco)

    L'application s'adapte à votre rôle grâce à son commutateur de profils :

    • Chef de projet (PM) : Vue opérationnelle classique.
    • Direction / Stratégique : Vue macro de type Dashboard avec indicateurs consolidés.
    • AppOps (Ressources) : Vue transversale pour filtrer par personne et voir l'ensemble de ses tâches en cours multi-projets.

    💡 Déploiement recommandé pour la Direction : Pour exploiter la vue Portfolio, il faut s'assurer que les projets de tous les PM soient stockés au même endroit. La méthode idéale consiste à créer un dossier racine partagé (ex: synchronisé via SharePoint / OneDrive local). Lors du clic sur "Ouvrir OneDrive", ciblez ce dossier racine : l'outil consolidera automatiquement tous les fichiers .json qu'il contient pour alimenter le dashboard consolidé de tous les projets en cours.

    🚀 1. Mise en route

    Pour que l'application puisse lire et écrire vos fichiers projets directement sur votre ordinateur, elle utilise des technologies Web modernes (File System Access API).

    📋 Prérequis

    • Utiliser un navigateur moderne basé sur Chromium (Google Chrome ou Microsoft Edge).
    • Avoir un dossier dédié sur votre ordinateur ou votre OneDrive pour y stocker les fichiers de vos projets.

    🏁 Premier démarrage

    1. Ouvrir l'espace de stockage : Cliquez sur le bouton "Ouvrir OneDrive" dans la barre latérale gauche. Votre navigateur va vous afficher une invite de sécurité : autorisez l'accès en lecture et écriture au dossier choisi.
    2. Créer votre premier projet : Dans le champ "Nom du projet...", saisissez le titre de votre projet puis cliquez sur le bouton ➕. Un fichier .json portant le nom de votre projet est instantanément créé dans votre dossier de travail.
    3. Sélectionner un projet : Vos projets s'affichent automatiquement dans la barre latérale, classés par statut (Cadrage, En cours, Terminés/Abandonnés). Cliquez sur l'un d'eux pour charger son tableau de bord.
    ⚠️ La Règle d'Or : L'application n'enregistre pas en temps réel pour préserver les performances. Pensez à cliquer sur le bouton vert "Enregistrer" en haut à droite avant de quitter ou de changer de projet !
