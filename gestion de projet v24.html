📘 Manuel Utilisateur - Gestionnaire de Projets Local Pro

Bienvenue dans votre outil de pilotage de projet local, sécurisé et boosté à l'IA. Cette application s'exécute entièrement dans votre navigateur tout en sauvegardant vos données directement dans votre écosystème local (OneDrive ou disque local). Pas de base de données complexe, pas de cloud tiers : vous êtes maître de vos données.

👑 0. Gestion des Profils Utilisateurs

L'application s'adapte à votre rôle grâce à son commutateur de profils situé dans la barre latérale :

  • Chef de projet (PM) : C'est la vue opérationnelle par défaut. Permet d'éditer l'intégralité d'un projet sélectionné dans la barre latérale.
  • Direction / Stratégique : Propose une vue macro de type Dashboard avec des compteurs consolidés pertinents (volume de projets actifs, budget cumulé, charge globale en JH, alertes bloquantes) filtrables par priorité de projet (P1, P2, P3), ainsi qu'un tableau d'exploration enrichi affichant les alertes et tâches bloquées par dossier.

🚀 1. Mise en route

Pour que l'application puisse lire et écrire vos fichiers projets directement sur votre ordinateur, elle utilise des technologies Web modernes (File System Access API).

📋 Prérequis

  • Utiliser un navigateur moderne basé sur Chromium (Google Chrome ou Microsoft Edge).
  • Avoir un dossier dédié sur votre ordinateur ou votre OneDrive pour y stocker les fichiers de vos projets.

🏁 Premier démarrage

  1. Ouvrir l'espace de stockage : Cliquez sur le bouton "Ouvrir OneDrive" dans la barre latérale gauche. Votre navigateur va vous afficher une invite de sécurité : autorisez l'accès en lecture et écriture au dossier choisi.
  2. Créer votre premier projet : Dans le champ "Nom du projet...", saisissez le titre de votre projet puis cliquez sur le bouton ➕. Un fichier .json portant le nom de votre projet est instantanément créé dans votre dossier de travail.
  3. Sélectionner un projet : Vos projets s'affichent automatiquement dans la barre latérale, classés par statut (Cadrage, En cours, Terminés/Abandonnés). Cliquez sur l'un d'eux pour charger son tableau de bord.
⚠️ La Règle d'Or : L'application n'enregistre pas en temps réel pour préserver les performances. Pensez à cliquer sur le bouton vert "Enregistrer" en haut à droite avant de quitter ou de changer de projet !

🏢 2. Gestion Administrative & Stratégique

L'application sépare les informations d'identification globale des informations orientées pour le management.

📋 Onglet : Description

  • Priorité Projet : Renseignez la typologie de priorité globale du projet (P1, P2 >50JH, ou P3 <50JH). Ce tag permet à la direction de filtrer l'ensemble du portefeuille d'un seul clic.
  • Codes et Alias : Renseignez le Code Projet, le code applicatif/produit (Format Fxx - nom_appli) et l'alias d'usage chez le client.
  • Historique CdP : En cliquant sur la case Chef de projet, un popover s'ouvre pour vous permettre d'ajouter le CdP actuel avec sa date de prise de poste.
  • Plannings Macro : Cliquez sur Planning Global pour renseigner graphiquement les dates initiales et révisées (Début/Fin).
  • Tableau des Participants : Listez vos ressources, leurs équipes, et leur disponibilité (en %).

🏢 Onglet : Contexte & Infos

Cet onglet est spécifiquement conçu pour faciliter le reporting auprès du management :

  • Client : Sélectionnez l'entité concernée dans le menu déroulant (CA-PFM Fr, CA-PFM Corp, CA-LF).
  • Champs textuels d'impact : Rédigez le contexte, le descriptif technique des flux, et surtout les impacts business pour le client.

🎬 3. Pilotage Opérationnel (Plan d'action & Risques)

C'est le cœur réactif de votre application. L'onglet Plan d'action & Risques vous permet de structurer l'exécution de vos tâches.

📑 Organisation par Streams (Chapitres)

  • Vous pouvez modifier l'ordre des chapitres entiers à l'aide des flèches 🔼 / 🔽 situées à droite des barres de titre de chaque Stream.
  • Les chapitres n'ayant aucune tâche "en cours" se replient automatiquement pour économiser l'espace visuel.

📥 Import / Export & Raccourcis

  • Import Excel Rapide : Vous avez déjà un tableau Excel ? Copiez simplement vos lignes (12 colonnes dans l'ordre du tableau) et collez-les directement dans la zone de texte pointillée. L'application parse le contenu instantanément.
  • Export CSV : Exportez votre plan d'action d'un clic pour l'envoyer par mail ou l'ouvrir dans Excel.
  • 👥 Synchronisation d'équipe : Cliquez sur "Ajouter équipes réalisatrices aux participants" pour injecter automatiquement toutes les équipes saisies dans vos actions directement dans votre liste de participants.

⏳ Moteur de Rétro-planning Avancé

Si vous avez correctement saisi des Prédécesseurs (ex: l'action ACT-02 dépend de l'ACT-01), l'application peut calculer votre rétro-planning :

  1. Saisissez une Date fin impérative client dans l'onglet Description.
  2. Cliquez sur le bouton rouge "Calculer le Rétro-planning".
  3. L'application calcule par propagation inverse les dates au plus tard pour toutes les tâches au statut "à démarrer".

🏄‍♂️ 4. Vues Agiles & Visuelles

L'application transforme vos données tabulaires en représentations graphiques interactives.

📋 Vue Kanban

  • Vos actions y sont représentées sous forme de cartes agiles réparties en 4 colonnes (À démarrer, En cours, Bloqué, Terminé).
  • Utilisez les boutons de transfert rapides ◀ / ▶ sur les cartes pour faire progresser une tâche d'une colonne à l'autre.

📅 Planning (Gantt)

  • Zoom adaptable : Basculez entre une vision par Jour, Semaine, ou Mois selon la durée du projet.
  • Chemin Critique : Les tâches qui conditionnent directement la date de fin du projet sont entourées d'un halo rouge lumineux.
  • Ligne du jour : Une ligne verticale pointillée rouge indique la date du jour pour visualiser instantanément votre retard ou votre avance.

📝 5. Notes & Comptes Rendus

Plus besoin d'ouvrir un bloc-notes externe pendant vos réunions.

  • Raccourcis de formatage : Utilisez les boutons de la barre d'outils (H1, H2, Gras, Liste) pour structurer votre texte.
  • Mode Prévisualisation : Basculez sur 👁️ Prévisualiser pour transformer instantanément vos notes Markdown en un rendu HTML propre.
  • Export texte : Cliquez sur "Exporter (.txt)" pour récupérer un fichier texte brut de votre compte rendu.
